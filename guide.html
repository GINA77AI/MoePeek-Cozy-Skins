✨ 功能清单
🔄
自动语言检测

通过 Unicode 范围自动识别中/英文，智能决定翻译方向，无需手动切换

👁️
macOS 原生通知

翻译结果以系统通知形式弹出，几秒后自动消失，不打断当前操作

⏸️
开关自由控制

菜单栏一键开启/暂停监听。不想翻译时关掉，完全静默无打扰

💰
免费零成本

使用 MyMemory 免费翻译 API，每日 1000 词免费额度，个人使用绰绰有余

🛡️
安全隐私

纯本地运行，代码完全开源可审查。仅发送需要翻译的文本片段到翻译 API

⌨️
